# Product

## Register

product

## Users

The public website serves prospective students and families. The admin dashboard serves a small number of authorised college staff who need to keep programme information accurate without editing code.

## Product Purpose

Present Utagba-Ogbe Technical College and its programmes clearly, while providing a simple protected workflow for updating course-list and course-detail content. Success means an authorised administrator can find a programme, edit its text or image, save once, and see the change on the public website.

## Brand Personality

Practical, credible, and welcoming.

## Anti-references

Avoid complex enterprise CMS patterns, decorative dashboards, dense configuration screens, major changes to the existing public website, and unfamiliar controls.

## Design Principles

1. Keep the public website visually stable.
2. Make the common editing task obvious and fast.
3. Use familiar form controls and explicit save feedback.
4. Protect administrative access and keep secrets server-side.
5. Preserve reliable static fallbacks when dynamic content is unavailable.

## Accessibility & Inclusion

Use keyboard-accessible controls, visible focus states, clear labels, readable contrast, descriptive status messages, and responsive layouts suitable for desktop and mobile administration.
